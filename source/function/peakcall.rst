peak calling
------------

CPT3 contains peak calling method with MACS2 (suggest) or SPP method.

.. code:: bash

    $ java -jar ChIA-PET.jar --macs2 macs2 \
    macs2 path, using macs2 callpeak to detect anchor peak with alignment file. default: N
